
# Pokemon Game
A simple game for Pokemon Fans testing Gen 1 Pokemon knowledge.


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd pokemon-game
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
